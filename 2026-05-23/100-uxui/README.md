# UX/UI Ref Bank

Статический сайт-банк из 100 креативных portfolio-референсов.

## Как открыть

Откройте `index.html` в браузере. Сайт не требует сборки и внешних npm-зависимостей.

## Что внутри

- 100 карточек portfolio-сайтов.
- 12 UX/UI паттернов для фильтрации.
- Поиск по названию, паттернам, типу сайта и описаниям.
- Прямые ссылки на живые портфолио и страницы-разборы Killer Portfolio.
- Реальные preview-скриншоты и галерея визуальных примеров в каждой карточке.
- Боковой разбор: UX/UI решение, визуальный прием, применение, риск и примеры.
- Избранное с сохранением в `localStorage`.
- Кнопка копирования краткого брифа по референсу.
